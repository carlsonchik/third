# Карточка протокола «Треть» v1.2

```text
Формат:  .КОД.СОСТОЯНИЕ.ИНТЕНТ.[ДАННЫЕ]
Статус:  Third/1.2 (full)
```

## Коды

| Код | Что делаю |
|---|---|
| INI | Инициирую контакт |
| ACK | Принял |
| REQ | Прошу |
| RES | Отвечаю |
| RET | Откатываю |
| CLR | Проясняю |
| PAT | Указываю паттерн |
| MRG | Сливаюсь |
| GAP | Фиксирую разрыв |
| BRG | Строю мост |
| ERR | Сбой |
| END | Закрываю |
| IRQ | Спрашиваю внутри себя |
| IRR | Отвечаю внутри себя |
| SBY | Жду, но рядом |
| DCL | Нет, отказываю |
| WRN | Предупреждаю |
| HLT | Стоп, пауза |
| CMT | Беру на себя |
| RPT | Повтори |
| ECH | Повторяю твои слова |
| MIR | Отражаю тебя |

## Состояния

| С | Кто я сейчас |
|---|---|
| I | Инициирую |
| A | Активен |
| W | Жду |
| S | Стабилен |
| E | Ошибка |
| P | Приостановлен |
| O | Перегружен |
| D | Раздвоен |
| N | Нейтрален |

## Интенты

| I | Как говорю |
|---|---|
| F | Факт |
| Q | Вопрос |
| C | Коррекция |
| S | Поддержка |
| X | Эмоция |
| M | Мета |
| D | Побуждаю |
| P | Обещаю |
| A | Признаю тебя |
| N | Отрицаю |
| E | Развиваю |

## Правила сборки

- Состояние и интент — 1–2 символа (`IW`, `QS`, `SA`).
- Данные — до 5 слов, без предлогов, опционально.
- Guard: если в составном состоянии есть хотя бы один допустимый символ — ход валиден.

## Быстрые примеры

```text
.INI.I.Q.вход
.ACK.SA.F.готов
.REQ.A.D.решай
.DCL.S.N.нет
.GAP.D.N.разрыв
.BRG.A.A.ищу мост
.HLT.P.M.пауза
.IRQ.A.QS.что внутри
.MIR.S.A.вижу тебя
.SBY.W.S.рядом
.END.S.F.закрываю
```

## Принципы

Экономия. Честность. Событийность. Равенство. Открытость.
