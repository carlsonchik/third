# Треть v1.2 — шпаргалка

---

## Формат сигнала

```
.КОД.СОСТОЯНИЕ.ИНТЕНТ.данные (до 5 слов)
```

Все четыре части опциональны. Минимальный сигнал: `.INI`. Полный: `.REQ.A.Q уточни срок и приоритет`.

---

## Коды

| Код | Значение |
|-----|----------|
| `ACK` | Подтверждение, принято |
| `BRG` | Попытка соединить расхождение |
| `CLR` | Прояснение |
| `CMT` | Принятие обязательства |
| `DCL` | Отказ, несогласие, отклонение |
| `ECH` | Эхо: повтор чужого сигнала для проверки понимания |
| `END` | Завершение процесса |
| `ERR` | Ошибка, сбой |
| `GAP` | Фиксация расхождения |
| `HLT` | Пауза, остановка процесса |
| `INI` | Инициация контакта |
| `IRQ` | Интроспективный запрос |
| `IRR` | Интроспективный ответ |
| `MIR` | Зеркало: отражение состояния собеседника без нового содержания |
| `MRG` | Слияние точек зрения |
| `PAT` | Указание на паттерн или повторяемость |
| `REQ` | Запрос действия или ответа |
| `RES` | Ответ на запрос |
| `RET` | Возврат, откат |
| `RPT` | Запрос на повтор |
| `SBY` | Активное ожидание |
| `WRN` | Предупреждение о нарастании напряжения или риска |

---

## Состояния

| Состояние | Значение |
|------------|----------|
| `I` | Инициирую |
| `A` | Активен |
| `W` | Жду |
| `S` | Стабилен |
| `E` | Ошибка или сбой |
| `P` | Приостановлен |
| `O` | Перегружен |
| `D` | Внутренне раздвоен |
| `N` | Нейтрален, вне активного включения |

---

## Интенты

| Интент | Значение |
|--------|----------|
| `F` | Факт, информация |
| `Q` | Вопрос |
| `C` | Коррекция |
| `S` | Поддержка |
| `X` | Эмоция, реакция |
| `M` | Мета-уровень, о протоколе |
| `D` | Побуждение к действию |
| `P` | Обязательство, обещание (commissive) |
| `A` | Признание позиции другого |
| `N` | Отрицание, несогласие |
| `E` | Elaboration, развитие без разрыва |

---

## Примеры сборки

```
.INI.I.F    — открываю диалог, сообщаю факт
.REQ.A.Q    — активно запрашиваю, жду ответа на вопрос
.ACK.W.C    — подтвердил получение, жду, уточняю
.GAP.S.M    — стабильно фиксирую разрыв, говорю о диалоге
.ERR.E.F    — ошибка, сообщаю факт о том, что пошло не так
.END.S.X    — завершаю стабильно, закрываю линию
.SBY.W      — здесь, жду, интент не важен
.HLT.P.X    — пауза, не готов сейчас вести диалог
.IRQ.A.Q    — интроспективный запрос, жду ответа
.IRR.D.F    — интроспективный ответ, внутренне раздвоен, сообщаю факт
.CMT.S.P    — стабилен, принимаю обязательство
.DCL.A.N    — активно отказываю, несогласие
```
